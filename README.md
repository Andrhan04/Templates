# Шаблоны для контестов

<h6>Все написано на классах</h6>
Алгоритмы:
1.   SQRT декомпозиция
2.   Дерево отрезков
3.   Дерево фенвика
4.   КМП(Кнут Морис Прат)
5.   Динамика по профилю(Паркет)
6.  Выпуклая оболочка
7.   Пара ближайших точек
8.   Дерево(реализация на нодах)
9.   Поиск точек сочленения(графы)
10.   Поиск мостов(графы)



Первоначальный посев частиц генерируется в буферную зонуАлгоритмы:
1. Выбираем произвольное число (от 1 до 999) - координата x
2. Выбираем произвольное число (от 1 до 999) - координата y
3. Проверяем, что нет частицы с такой координатой. Если нет, то записываем в посев.
